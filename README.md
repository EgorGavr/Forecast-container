# Forecast-Container
Простой класс: Суточный прогноз — содержит информацию о дате прогноза (например,
unix time stamp), температуре воздуха утром, днём и вечером (в градусах цельсия, не может
быть меньше −273), погодном явлении (солнечно, облачно, дождь или снег, если было
несколько явлений то выбирается самое плохое) и количестве осадков (в мм). Прогноз
считается ошибочным, если при погодных явлениях «солнечно» или «облачно» количество
осадков не равняется 0 мм, если снег прогнозируется при температурах выше 0°C, если
любая из температур выходит за пределы [−100 , +60] °C или если количество осадков
превышает 1500 мм.
Методы простого класса (помимо общих):
• создание экземпляра класса с инициализацией датой, температурами, погодным
явлением и количеством осадков;
• создание экземпляра класса с инициализацией только датой, температурой и количеством
осадков (погодное явление нужно выбрать автоматически, исходя из остальных
параметров);
• (+=) объединение двух прогнозов за одни и те же сутки (для погоды и осадков выбирается
среднее, для явления — наихудшее из двух);
• получение средней температуры за сутки;
• проверка (да/нет), является ли данных прогноз ошибочным;
• (<=>) сравнение двух экземпляров класса по дате прогноза.
Сложный класс: Прогноз погоды – определяется вектором суточных прогнозов.
Методы сложного класса (помимо общих):
• создание экземпляров класса с инициализацией заданным количеством суточных
прогнозов из массива прогнозов;
• создание экземпляров класса с инициализацией одним суточным прогнозом;
• (+=) добавление нового суточного прогноза;
• ([]) получение суточного прогноза по его номеру (возврат по ссылке);
• удаление суточного прогноза по его номеру;
• поиск самого холодного дня за указанный промежуток;
• поиск ближайшего солнечного дня для текущей даты;
• удаление из вектора всех ошибочных прогнозов;
• объединение всех повторяющихся прогнозов за одни и те же сутки в один;
• упорядочивание суточных прогнозов по дате;
• получение подмножества суточных прогнозов, упорядоченных по дате, за заданный
месяц.
